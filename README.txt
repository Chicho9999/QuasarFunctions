Proyecto realizado usando tecnologias del framework de .Net core con Azure Functions (Serverless) y hosteada en Azure. 

Para acceder a la API correr las siguientes URLs: 

POST: https://fuegodequasarfunctions.azurewebsites.net/api/topsecret

POST: https://fuegodequasarfunctions.azurewebsites.net/api/topsecret_split?name={name}

GET: https://fuegodequasarfunctions.azurewebsites.net/api/topsecret_split?name={name}

Para correrlo en local, ejecutar el archivo FuegoDeQuasarFunctions.sln, restaurar los paquetes de nuget si es necesario e inicializar tocando el boton F5 o haciendo Click en
